# 가이드 콘텐츠 편집 방법

## 파일 위치
- **가이드 데이터**: `lib/guideData.js`
- **가이드 이미지**: `public/images/[카테고리폴더]/[가이드폴더]/`

## 현재 구조

### 카테고리
| ID | 제목 | 가이드 수 |
|---|---|---|
| `terminal` | 단말기 / 결제 | 5개 |
| `closing` | 마감 / 재오픈 | 1개 |

### 가이드 목록
**단말기 / 결제**
- 긴 영수증이 안 나와요 (단축키: 0 > 1 > 0 > 1)
- 호스트 영수증 출력 (단축키: 3 > 1 > 1 > 입력 2번)
- "영수증 출력 확인" 메시지 (단축키: 0 > 0 > 000)
- CAT 단말기 연결 오류
- 재고 출력이 안 돼요

**마감 / 재오픈**
- 마감 후 손님이 왔을 때

## 새 가이드 추가하는 법

### 1. 이미지 준비
`public/images/` 아래에 폴더를 만들고 스텝별 사진을 넣습니다.
```
public/images/[카테고리폴더]/[가이드폴더]/step1.jpg
public/images/[카테고리폴더]/[가이드폴더]/step2.jpg
...
```

### 2. guideData.js에 가이드 추가
해당 카테고리의 `guides` 배열에 아래 형식으로 추가:

```js
{
  id: '고유-영문-id',           // URL용, 중복 불가
  title: '가이드 제목',          // 카드에 표시되는 제목
  symptom: '증상 설명',          // 어떤 상황인지 한 줄 설명
  keywords: ['검색키워드1', '검색키워드2'],  // 통합 검색에 잡힐 키워드
  shortcut: '0 > 1 > 2',       // 단축키 (없으면 null)
  steps: [
    {
      action: '스텝 제목',       // 볼드로 표시됨
      detail: '상세 설명',       // 아래 작은 글씨
      image: '/images/폴더/step1.jpg'  // 이미지 (없으면 null)
    },
    // ... 스텝 반복
  ],
  note: '주의사항'              // 하단 안내 메시지 (없으면 null)
}
```

### 3. 새 카테고리 추가하는 법
`categories` 배열에 추가:

```js
{
  id: '고유-영문-id',
  title: '카테고리 제목',
  icon: '이모지(현재 미사용, SVG로 대체됨)',
  desc: '카테고리 설명',
  color: '#4ECDC4',
  gradient: 'linear-gradient(135deg, #4ECDC4, #7eddd6)',
  notice: '카테고리 상단 안내 (없으면 생략)',
  guides: [ ... ]
}
```

새 카테고리 추가 시 `app/page.js`의 `CATEGORY_ICONS` 객체에도 SVG 아이콘을 추가해야 합니다.

## 클로드에게 요청하는 법
사진과 함께 아래처럼 말하면 됩니다:

> "이 사진들로 새 가이드 만들어줘"
> - 제목: OOO
> - 증상: OOO
> - 카테고리: 단말기/결제 (또는 새 카테고리)
> - 스텝 1: (사진) OOO 하세요
> - 스텝 2: (사진) OOO 하세요
> - 주의사항: OOO

사진만 주고 "이거 순서대로 가이드 만들어줘"라고 해도 됩니다.

## 이미지 임시 폴더
새 가이드용 사진은 먼저 여기에 넣어주세요:
```
public/images/_새가이드/
```
클로드가 가이드 작성 시 적절한 폴더로 옮깁니다.
